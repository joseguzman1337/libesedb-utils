Extensible Storage Engine DB access library — packaging metadata for the
Debian/Ubuntu `libesedb-utils` package.

The authoritative library lives at <https://github.com/libyal/libesedb>.
This repository only tracks the packaging metadata: `LICENSE`,
`libesedb.md`, `project.json`, `.whitesource`, `renovate.json`, and the
GitHub workflows that lint, score, and merge it.

See [`SECURITY.md`](./SECURITY.md) for the security policy.
